Figures from SNV analysis for days post infection to understand the distribution of variants on each day post infection. 

Contents:

1) all_samples_dpi.pdf = for all variants, the number of samples with variant data on each day post infection

2) ns_samples_dpi.pdf = for nonsynonymous sites, the number of samples with variant data on each day post infection

3) all_variants_dpi.pdf = for all variants, the number of variants across all samples on each day post infection

4) ns_variants_dpi.pdf = for nonsynonymous sites, the number of variants across all samples on each day post infection

5) ns_mean_variants_sample.pdf = for nonsynonymous sites, the mean number of variants across samples on each day post infection
